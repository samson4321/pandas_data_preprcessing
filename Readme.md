# Pandas Data Preprocessing from URL

This project demonstrates a basic data preprocessing workflow using **pandas**.

## Steps performed
1. Load a dataset directly from an online URL
2. Inspect the data structure
3. Add missing column headers
4. Save the cleaned dataset to a local CSV file

- Source: UCI Machine Learning Repository (Automobile dataset)
